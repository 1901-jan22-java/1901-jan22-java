The Display Property

The display property — Standard values such as block, inline or inline-block can change how elements behave in normal flow

Block

Starts on a new line, and takes up the whole width



Inline
Inline

Displays an element as an inline element. Any height and width properties will have no effect

inline-block

Now let's look at the Flex Box

The main idea behind the flex layout is to give the container the ability to alter
its items' width/height (and order) to best fill the available space
(mostly to accommodate to all kind of display devices and screen sizes).
A flex container expands items to fill available free space, or shrinks them to prevent overflow.

Lastly, let's see what the grid layout does

Like tables, grid layout enables an author to align elements into columns and rows.
However, many more layouts are either possible or easier with CSS grid than they were with tables.
For example, a grid container's child elements could position themselves so they actually overlap and layer, similar to CSS positioned elements.
