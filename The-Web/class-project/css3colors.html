New Colors in CSS 3

  • We no longer just have to use rgb or hexadecimal values for colors css3 comes with 147 keywords that are colors

  • We now have RGBA instead of RGB so we can add alpha to colors making them opaque

  • HSL(hue, saturation, lightness) is now in css3 allowing us manipulate the saturation or brightness of colors while keeping the same hue
