DOM Manpulation

The Document Object Model is a cross-platform and language independent application programming interface that treats an HTML, XHTML, or XML document as a tree structure wherein each element is an object representing a part of the document. In simpler terms, the DOM is a tree-like structure representing the elements in your HTML page. We use JS to dynamically alter these elements via the global document object.
