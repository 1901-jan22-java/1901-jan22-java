Now let's look at the Flex Box

The main idea behind the flex layout is to give the container the ability to alter
its items' width/height (and order) to best fill the available space
(mostly to accommodate to all kind of display devices and screen sizes).
A flex container expands items to fill available free space, or shrinks them to prevent overflow.
