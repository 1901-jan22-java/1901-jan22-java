This is a description of what HTML 5 is and how it differs from previous versions of HTML.

HTML 5 both introduces and removes detailing present in previous versions.



Introduction

HTML stands for HyperText Markup Language. It provides properties and behaviors of web page content by implementing a markup based pattern to it. Prior to being updated to the latest iteration, HTML had not been updated since the year 2000. Webpages required the use of external plug-ins in order to perform certain functionality like playing video media.

HTML 5 was released in 2008 with the goal to simplify and improve upon previous version of HTML by introducing support for new multimedia as well as simplifying the language to make it more readable to humans without losing intrepretation by a machine.

With HTML 5, new features could be added while remaining backwards-compatible with older software. Because of this, adoption of HTML 5 was very large from the start.

The adoption of HTML 5 also allowed for media to be played without the need for external plug-ins or helper software. This allowed for a more streamlined user experience.

    Coding Practices


    HTML 5 allows for some practices that arent considered the best

    • Elements should be in all lower case
    • Attributes names should also be all lower case
    • When using =. Put whatever you are assigning in quotes and dont leave a space in between the = and "
    • Elements dont have to be closed but they ALWAYS SHOULD
    • When dealing with images always add the height and width and use the alt attribute
    • You can ommit the html, body, and head tag but it is not recommended
