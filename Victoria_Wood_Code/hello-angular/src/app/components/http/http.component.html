Understanding HTTP Requests in Angular

Most front end applications communicate with backend services over HTTP. One common approach to asynchronous requests an responses with Angular is to use Observables and the HttpClientModule.

So what are Observables?

Observables are arrays whose items arrive asynchronously over time. Observables help you manage asynchronous data, such as data coming from a backend service. Observables are used within Angular itself, including its event system and http client service. To use observables, Angular uses a third-party library called Reactive Extensions (RxJs).
